### 「機械学習基礎」

機械学習の全体像および基本的な手順を学んだ後、基礎的な識別手法および回帰手法を学びます。GUIで操作ができるマイニングツールWekaで機械学習の手順や主要なパラメータを確認し、Pythonで機械学習ライブラリScikit-learnを使用して実習を行います。
